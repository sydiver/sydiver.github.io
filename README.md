# GitHub Pages (TeXt theme starter)

This repository is a starter for a personal website on GitHub Pages using the **TeXt** Jekyll theme.

## Quick start (GitHub Pages)
1. Create a new repo named: `<your-github-username>.github.io`
2. Upload everything from this folder to the repo root (or drag-and-drop).
3. Go to **Settings â†’ Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** (or master) / **root**
4. Wait for the build, then open: `https://<your-github-username>.github.io`

## Edit content
- Home page: `index.html`
- About page: `about.md`
- Archive: `archive.html`
- Posts: `_posts/YYYY-MM-DD-title.md`

## Math (LaTeX)
MathJax is enabled globally in `_config.yml`.
Use inline: `$E=mc^2$`
Use block:
```tex
$$\nabla \cdot \mathbf{u} = 0$$
```
